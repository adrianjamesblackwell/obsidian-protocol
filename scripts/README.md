`docs/walkthrough.md`, you are expected to write your own exploit code
here (the VECTOR-I path traversal/RCE script, the VECTOR-II PwnKit
build/run steps).

Suggested files (for you to create):
- `vector1_exploit.py` — VECTOR-I: path traversal + RCE (CVE-2021-41773/42013)
- `vector2_pwnkit.c` or `.sh` — VECTOR-II: PwnKit local priv-esc (CVE-2021-4034)

Writing these yourself, in your own understanding, is what lets the
repo say "I ran this and understood it" instead of "I just copied a
tutorial" — the first question in an interview about this project
will likely be "how did you write this code."
